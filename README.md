HOW TO RUN THE CODE:

Go to the green CODE drop down (Make sure you are exactly in the main branch, and not selected any file).

Select code spaces, and go to shiny space winner. Go to extensions, and download the Live server add on (just search live server and select the first one).

Wait a couple seconds and then a button on the bottom right corner will appear that says Go live. Press that and your all set :D


Also, to make the code be up-to-date, go to the terminal and run "git pull origin main"
